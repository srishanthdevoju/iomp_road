System Modules

6 modules →
🔍

Object Detector

detector.py

YOLOv8 nano model detects persons, bicycles, cars, motorcycles, buses, and trucks with 0.5 confidence threshold.

📡

Centroid Tracker

tracker.py

Euclidean distance matching across frames with 30-frame history. Detects approaching, departing, or stationary motion.

🛤️

Lane Detector

lane_detector.py

Divides frame into three zones — Left (0-35%), Center (35-65%), Right (65-100%) for spatial lane classification.

⚡

Risk Assessor

risk_assessor.py

Composite risk score (0-100%) using 4 weighted factors: distance, motion, lane position, and object type.

🎨

Visualizer

visualizer.py

Color-coded bounding boxes (green→red), motion labels, lane dividers, stats dashboard, and critical warnings.

⚙️

Configuration

config.py

Central hub for all tunable parameters including model path, thresholds, weights, colors, and output settings.

📹 Object Detection — YOLOv8

The YOLOv8 nano model performs real-time inference on each frame, detecting 6 road-relevant classes: person, bicycle, car, motorcycle, bus, and truck. Each detection includes bounding box coordinates, class label, confidence score, center point, and area.
